## vue2+puppeteer+node输入网址，点击按钮把网页转为pdf

## 运行vue项目
#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run dev
```

## 运行node项目，把服务跑起来

```
cd server
node index
```


#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
